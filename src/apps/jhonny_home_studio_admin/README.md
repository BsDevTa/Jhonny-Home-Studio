# Jhonny Home Studio Admin

Painel administrativo Angular para gerenciamento do catálogo do Jhonny Home Studio.

## Requisitos

- Node.js
- npm
- API local em `http://localhost:5299`

## Executar

```powershell
cd "C:\Users\FOLHA_USER\Documents\Jhonny Home Studio\src\apps\jhonny_home_studio_admin"
npm install
npm run start
```

Abra `http://localhost:4200`.

## Login de desenvolvimento

- E-mail: `admin@jhonnyhomestudio.com`
- Senha: `Admin@123456`

## Recursos da Etapa 7A

- Login administrativo com JWT
- Dashboard do catálogo
- CRUD de categorias
- CRUD de serviços
- Agenda administrativa com filtros, ações rápidas e detalhe
- Consulta de clientes com endereços e histórico de agendamentos
- Ativação e desativação de categorias e serviços
- Layout responsivo para desktop e mobile

Clientes, stories e pagamentos permanecem fora desta etapa.

npm run build
